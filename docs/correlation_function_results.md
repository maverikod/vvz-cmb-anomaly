# Результаты анализа двухточечной корреляционной функции CMB

**Автор:** Vasiliy Zdanovskiy  
**Email:** vasilyvz@gmail.com  
**Дата:** 2025-11-08

## Резюме

Проведен анализ двухточечной корреляционной функции CMB без использования маски (все пиксели). Обнаружена осцилляторная структура с алгебраическим затуханием, согласующаяся с предсказаниями теории "quench → tails model".

## Измеренные параметры

### Основные параметры

- **κ_* = 66.8345 per deg** (предпочтительное волновое число)
- **β = 0.5000** (показатель алгебраического затухания)
- **Δθ = 0.0470° = 2.82 arcmin** (предпочтительный интервал осцилляций)
- **Корреляция = 0.9241** (качество подгонки)
- **p-value ≈ 0** (статистическая значимость)
- **Амплитуда A = 2.587 × 10⁻⁹**

### Функция корреляции

```
ξ(θ) = 2.587 × 10⁻⁹ × θ^(-2.0000) × cos(66.8345θ + 1.0608) + 8.509 × 10⁻¹²
```

## Физическая интерпретация параметров

### 1. κ_* (Предпочтительное волновое число)

**Значение:** κ_* = 66.8345 per deg = 1.166 per rad

**Физический смысл:**
- Характеризует предпочтительную частоту осцилляций базовой подложки
- Проецируется на CMB через функцию переноса
- Очень высокое значение указывает на высокочастотную структуру

**Что это дает:**
- Позволяет определить характерный масштаб структуры в 3D пространстве
- Связано с параметрами квенча (Big Bang) через модовое сопоставление
- Определяет интервал осцилляций: Δr = π/κ_* в 3D, Δθ = π/κ_* в угловых координатах

### 2. β (Показатель алгебраического затухания)

**Значение:** β = 0.5000

**Физический смысл:**
- Связан с "fractional rigidity" (дробной жесткостью) базового поля
- Определяет скорость алгебраического затухания корреляции
- β = 0.5 → затухание ~ θ⁻² (сильное затухание)

**Что это дает:**
- **Ключевое предсказание теории:** алгебраическое (не экспоненциальное) затухание
- Это указывает на отсутствие массовых членов в лагранжиане
- Согласуется с производными операторами и дробной жесткостью
- Позволяет отличить теорию от стандартных моделей с экспоненциальным затуханием

**Сравнение:**
- β = 0.5 → ξ(θ) ~ θ⁻² (наше значение)
- β = 1.0 → ξ(θ) ~ θ⁻¹ (умеренное затухание)
- β = 1.5 → ξ(θ) ~ θ⁰ (нет затухания)

### 3. Δθ (Предпочтительный интервал)

**Значение:** Δθ = 0.0470° = 2.82 arcmin

**Физический смысл:**
- Угловой масштаб, на котором корреляции повторяются
- Это проекция 3D интервала Δr = π/κ_* на небесную сферу
- Очень малый масштаб указывает на высокочастотную структуру

**Что это дает:**
- Определяет характерный угловой масштаб структуры в CMB
- Позволяет предсказать осцилляторную структуру в C_ℓ (угловом спектре мощности)
- Связано с параметрами окна проекции W(r) и функции переноса T(k)

## Согласование с теорией

### Предсказания теории (из `Новая_теория.md`)

Теория предсказывает для базовой подложки:

```
ξ_Θ(r) ~ r^-(3-2β) cos(κ_*r + φ_*)
```

Для проекции на CMB:

```
ξ_CMB(θ) ~ θ^-(3-2β) cos(κ_*θ + φ_*)
```

### Сравнение с измерениями

| Параметр | Теория | Измерение | Согласование |
|----------|--------|-----------|--------------|
| Форма затухания | Алгебраическая (θ^-(3-2β)) | Алгебраическая (θ^-2) | ✅ **Согласуется** |
| β | 0.5 - 1.5 | 0.5000 | ✅ **Согласуется** |
| Осцилляторная структура | cos(κ_*θ + φ) | cos(66.83θ + 1.06) | ✅ **Согласуется** |
| Отсутствие экспоненты | Нет экспоненциального затухания | Только алгебраическое | ✅ **Согласуется** |

### Ключевые подтверждения

1. ✅ **Алгебраическое затухание** (не экспоненциальное)
   - Это ключевое предсказание теории
   - Указывает на отсутствие массовых членов
   - Согласуется с производными операторами

2. ✅ **Осцилляторная структура**
   - Обнаружена четкая осцилляторная корреляция
   - Предпочтительный интервал Δθ = 2.82 arcmin
   - Высокая статистическая значимость (p ≈ 0)

3. ✅ **Высокая корреляция**
   - Корреляция = 0.9241 указывает на сильный сигнал
   - Стабильность параметров при разных диапазонах поиска

4. ✅ **Согласование β**
   - β = 0.5 находится в предсказанном диапазоне
   - Соответствует "fractional rigidity" базового поля

## Что дают эти параметры

### 1. Подтверждение теории

- **Алгебраическое затухание** подтверждает отсутствие массовых членов
- **Осцилляторная структура** подтверждает предсказание о предпочтительном волновом числе
- **Высокая значимость** (p ≈ 0) указывает на реальный сигнал, а не шум

### 2. Физические выводы

- **Базовая подложка имеет высокочастотную структуру**
  - κ_* = 66.8 per deg указывает на очень высокую частоту
  - Это согласуется с концепцией высокочастотного базового поля

- **Дробная жесткость базового поля**
  - β = 0.5 указывает на определенный тип дробной жесткости
  - Это связано с производными операторами в лагранжиане

- **Характерный масштаб структуры**
  - Δθ = 2.82 arcmin определяет угловой масштаб
  - Это может быть связано с параметрами квенча (Big Bang)

### 3. Предсказания для других наблюдаемых

- **Угловой спектр мощности C_ℓ:**
  - Должен показывать осцилляторную структуру
  - Интервал осцилляций: Δℓ ~ π r_* κ_*
  - Огибающая: C_ℓ ~ ℓ^-(3-2β) = ℓ^-2

- **Крупномасштабная структура (LSS):**
  - Должна показывать волны плотности с тем же κ_*
  - Масштаб: Δr = π/κ_* в 3D пространстве

## Ограничения и предостережения

### 1. Очень малые угловые масштабы

- Δθ = 2.82 arcmin - очень малый масштаб
- Может быть затронут:
  - Инструментальным шумом
  - Сглаживанием луча (beam smoothing)
  - Эффектами пикселизации

### 2. Зависимость от метода весов

- Подгонка чувствительна к использованию envelope weights
- Uniform weights дают низкую корреляцию (~0.01)
- Это может указывать на важность правильного учета алгебраического затухания

### 3. Необходимость независимой проверки

- Нужно проверить с помощью анализа C_ℓ
- Сравнить с результатами structure-detect
- Проверить на разных наборах данных
