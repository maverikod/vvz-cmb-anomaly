# ЗАДАНИЕ: Соотнести кластеры аномалий с крупномасштабными структурами и процессами Галактики

1. Для каждого масштаба (1°, 5°, 25°):
    - Построить распределение аномалий по широте (b): гистограмма и среднее |b|.
    - Построить распределение по долготе (l): гистограмма, выявить концентрации.

2. Определить, какая доля аномалий находится в пределах:
    - |b| < 10° (тонкий диск)
    - 10° < |b| < 30° (толстый диск/облака)
    - |b| > 30° (гало/окологалактическая среда)

3. (По возможности) Сравнить позиции аномалий с картами пыли, HI, CO, магнитных полей.
    - Для этого можно скачать/использовать карты Planck dust, HI, CO.
    - Проверить, совпадают ли координаты аномалий с максимумами или “особенностями” этих карт.

4. Описать, какие масштабные процессы (газовые рукава, вспышки, суперпузыри, остатки SNR, магнитные структуры) могут быть причиной обнаруженных кластеров аномалий на каждом масштабе.

5. Сделать визуализацию: наложить аномалии на карту Галактики (например, на фон Planck dust/HI/CO).

6. Оформить выводы: Какие типы аномалий и на каких масштабах связаны с Галактикой и её процессами?
