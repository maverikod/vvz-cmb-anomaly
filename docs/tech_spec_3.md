# ЗАДАНИЕ: Проверка дискретизации масштабов и кластеризация аномалий по теоретическим уровням (без применения маски)

**Цель:**  
Проверить, распределяются ли найденные аномалии по радиусу в соответствии с теорией дискретных (разрешённых) масштабов среды. Если теория верна, то число аномалий должно “кластеризоваться” на определённых радиусах (логарифмически равномерных), а не распределяться равномерно или случайно.

---

## Входные данные:
- Таблица/датасет с аномалиями (минимум: радиус, координаты центра, “аномальность”).
  - Например, `results_anomalies_multi.csv` с колонкой `radius_deg`.

## Важно:
- **Маску применять НЕ нужно.**  
  Анализ проводится по всему небу, включая область Галактики и любые “грязные” зоны. Это принципиально, так как цель — исследование среды в целом, а не только “чистого” космологического сигнала.

## Теоретическая гипотеза:
- Устойчивые аномалии среды возникают только на дискретных масштабах:
  \[
  r_i = r_0 \cdot 10^{\alpha i},\quad \alpha \sim 0.5-1
  \]
  где $r_0$ — минимальный радиус в анализе, $i$ — целое число, $\alpha$ — шаг по логарифму радиуса.

---

## Пошаговый алгоритм:

1. **Взять все значения радиусов аномалий из датасета.**

2. **Вычислить логарифм радиуса:**  
   \[
   \log_{10} r
   \]
3. **Для заданного $\alpha$ (например, 0.5, 0.7 или 1.0):**
   - Привязать каждый радиус к ближайшему дискретному уровню $r_i$ по формуле:
     \[
     i = \text{round} \left( \frac{\log_{10}(r / r_0)}{\alpha} \right)
     \]
     \[
     r_i = r_0 \cdot 10^{\alpha i}
     \]
   - Записать для каждой аномалии ее “дискретный класс” ($i$ или $r_i$).

4. **Построить гистограмму/распределение числа аномалий по $r_i$.**
   - (Опционально: проверить результат для разных значений $\alpha$.)

5. **Сравнить распределение с равномерным или гладким (шумовым) распределением.**
   - Есть ли “пики” — избыток аномалий на определённых $r_i$?

6. **Вывести список радиусов $r_i$, на которых число аномалий максимальное.**

7. **Сделать краткий вывод:**
   - Поддерживается ли гипотеза о дискретных масштабах среды?
   - Насколько чётко видны “кластерные” уровни?
   - Виден ли “ступенчатый” характер распределения по масштабам?

---

## Формат отчёта:
- Краткая таблица или гистограмма с числом аномалий на каждом $r_i$.
- График или диаграмма для визуализации кластеризации по масштабу.
- Текстовый вывод о наличии (или отсутствии) дискретизации масштабов.
