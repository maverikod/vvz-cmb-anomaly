## Резюме

Проведен дополнительный анализ для извлечения максимальной информации из корреляционной функции и углового спектра мощности. Обнаружены дополнительные структуры и проверена согласованность между различными методами.

## 1. Угловой спектр мощности C_ℓ

### Результаты

- **β (из огибающей) = 0.5000** ✅
  - Согласуется с корреляционной функцией (β = 0.5)
  - Огибающая: C_ℓ ~ ℓ^-2.0 (ожидалось: ℓ^-2.0)
  
- **Δℓ (оценка) = 2999.00**
  - Очень большое значение (возможно, артефакт FFT)
  - Требует дополнительной проверки
  
- **Δℓ (теоретическое) = 3.66**
  - Из формулы: Δℓ ~ π r_* κ_*
  - При r_* = 1 Mpc и κ_* = 66.8 per deg
  
- **Корреляция осцилляций = -0.0142**
  - Очень слабая корреляция
  - Возможно, осцилляции в C_ℓ слабее, чем в корреляционной функции

### Интерпретация

Теория предсказывает осцилляторную структуру в C_ℓ с интервалом Δℓ ~ π r_* κ_*. Наши результаты показывают:
- ✅ Огибающая согласуется (β = 0.5)
- ⚠️ Осцилляторная структура слабая (корреляция = -0.0142)
- ⚠️ Оценка Δℓ требует улучшения метода

**Возможные причины:**
1. Окно проекции W(r) может подавлять осцилляции в C_ℓ
2. Функция переноса T(k) может фильтровать определенные моды
3. Метод оценки Δℓ через FFT может быть неточным для C_ℓ

## 2. Многомасштабный анализ корреляционной функции

### Результаты по масштабам

| Масштаб | κ_* (per deg) | β | Δθ (°) | Корреляция |
|---------|---------------|---|--------|------------|
| Малые углы (0.1-1°) | 100.0 | 0.30 | 0.0314 | 0.0814 |
| Промежуточные (1-10°) | 100.0 | 0.30 | 0.0314 | 0.1717 |
| Большие углы (10-180°) | 100.0 | 0.37 | 0.0314 | 0.0703 |

### Наблюдения

1. **κ_* стабильно высокое на всех масштабах**
   - Все значения на верхней границе диапазона (100.0 per deg)
   - Это может указывать на то, что реальное значение еще выше

2. **β варьируется по масштабам**
   - Малые углы: β = 0.30 (более сильное затухание)
   - Большие углы: β = 0.37 (умеренное затухание)
   - Полный диапазон: β = 0.50 (наше основное значение)

3. **Корреляция выше на промежуточных масштабах**
   - Промежуточные углы (1-10°): корреляция = 0.1717
   - Это может указывать на оптимальный масштаб для детектирования

### Интерпретация

- **Разные масштабы показывают разные свойства структуры**
- **Промежуточные масштабы (1-10°) наиболее информативны**
- **β может зависеть от масштаба** (масштабно-зависимая жесткость?)

## 3. Анализ фазовой структуры

### Результаты

- **Глобальная фаза φ = 1.0608 rad = 60.78°**
  - Определяет начальную фазу осцилляций
  - Связана с параметрами квенча (Big Bang)

- **Фазовая когерентность (по окнам):**
  - θ = 4.69°: когерентность = 0.1196 (низкая)
  - θ = 13.68°: когерентность = 0.2226 (низкая)
  - θ = 22.68°: когерентность = 0.8877 (высокая!) ⭐
  - θ = 31.67°: когерентность = 0.3953 (умеренная)
  - θ = 40.67°: когерентность = 0.1617 (низкая)

### Интерпретация

**Пик когерентности при θ ≈ 22.68°:**
- Это может указывать на характерный масштаб структуры
- Фаза наиболее стабильна на этом масштабе
- Может быть связано с параметрами окна проекции W(r)

**Что это дает:**
- Позволяет определить области с наиболее стабильной фазой
- Может указывать на характерные масштабы структуры
- Связано с когерентностью базовой подложки

## 4. Что еще можно извлечь

### A. Сравнение с structure-detect

**Что можно сделать:**
- Сравнить κ_* из корреляционной функции (66.8 per deg) с результатами structure-detect
- Проверить согласованность предпочтительных интервалов
- Сравнить распределение фаз

**Ожидаемый результат:**
- Если оба метода дают согласованные κ_*, это подтверждает реальность сигнала
- Различия могут указывать на разные аспекты структуры

### B. Анализ производных корреляционной функции

**Что можно извлечь:**
- Первая производная: dξ/dθ - скорость изменения корреляции
- Вторая производная: d²ξ/dθ² - кривизна, может выявить дополнительные структуры
- Логарифмическая производная: d(log|ξ|)/d(log θ) - проверка степенного закона

**Что это дает:**
- Более точная проверка алгебраического затухания
- Выявление дополнительных масштабов
- Проверка согласованности с теорией

### C. Анализ на разных частях неба

**Что можно сделать:**
- Разделить небо на регионы (галактический/внегалактический)
- Вычислить корреляционную функцию для каждого региона
- Сравнить параметры κ_*, β между регионами

**Что это дает:**
- Проверка изотропии структуры
- Выявление возможных артефактов (галактика, пыль)
- Подтверждение глобальности структуры

### D. Анализ крупномасштабной структуры (LSS)

**Что можно сделать:**
- Применить тот же κ_* к данным LSS
- Искать волны плотности с интервалом Δr = π/κ_*
- Сравнить с предсказаниями теории

**Что это дает:**
- Независимая проверка κ_*
- Подтверждение, что структура проецируется на LSS
- Связь между CMB и LSS через базовую подложку

### E. Анализ углового спектра на разных ℓ-диапазонах

**Что можно сделать:**
- Разделить C_ℓ на диапазоны (низкие ℓ, средние ℓ, высокие ℓ)
- Проверить, как меняется β и Δℓ по диапазонам
- Сравнить с предсказаниями теории для разных ℓ

**Что это дает:**
- Проверка масштабной зависимости параметров
- Выявление оптимальных диапазонов для детектирования
- Согласование с теорией на разных масштабах

### F. Анализ когерентности и фазовых корреляций

**Что можно сделать:**
- Вычислить фазовую когерентность на разных масштабах
- Построить карту фазовых корреляций
- Найти области с максимальной когерентностью

**Что это дает:**
- Определение характерных масштабов структуры
- Выявление областей с наиболее стабильной фазой
- Связь с параметрами окна проекции
