Постановка задачи: второй этап анализа по пыли

Цель:
Выявить морфологические и статистические особенности распределения пыли внутри и вокруг CMB-анoмалий, чтобы понять, существуют ли характерные границы, контрасты или структурные особенности, связанные с фазовой самоорганизацией среды.
Задача:

    Для каждой CMB-анoмалии (задан центр, радиус):

        Вычислить не только среднее, но и стандартное отклонение, максимальное/минимальное, перцентиль (например, 10%, 90%) значения пыли внутри радиуса аномалии.

        Оценить градиент пыли на границе аномалии (сравнить среднее значение в тонком кольце “border” вокруг аномалии и в центре).

    Построить профили:

        Построить усреднённый радиальный профиль пыли (dust profile) относительно центра аномалий: от 0 до 2R с шагом, усредняя по всем аномалиям одного масштаба.

        Построить аналогичный профиль для случайных (контрольных) точек.

    Сравнить “контраст” пыли:

        Рассчитать для каждой аномалии:
        контраст=mean dust (border)−mean dust (center)mean dust (center)
        контраст=mean dust (center)mean dust (border)−mean dust (center)​

        Построить распределение контраста для аномалий и контроля.

    Статистика по зонам:

        Повторить анализ отдельно для зон: тонкий диск, толстый диск, гало.

        Оценить, есть ли зоны, где аномалии “живут” на резких переходах плотности пыли.

Критерии и итог:

    Если аномалии CMB соответствуют резким границам пылевых структур, это поддерживает сценарий формирования дефектов на переходах фаз среды.

    Если аномалии “живут” только в идеально чистых/плавных зонах — это другой сценарий самоорганизации.

    Любая устойчивая разница между аномалиями и контрольными точками — ценный сигнал.
