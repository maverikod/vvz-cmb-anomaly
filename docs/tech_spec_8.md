Задача: фазовый морфологический анализ CMB-анoмалий с учётом газа (HI/CO) и пыли

Цель:
Проверить, совпадают ли морфологические и статистические свойства нейтрального водорода (HI), молекулярного газа (CO) и пыли в зонах CMB-анoмалий, и выявить признаки фазовой самоорганизации среды в разных компонентах ISM.
1. Для каждой CMB-анoмалии (заданы l, b, радиус):

    Извлечь радиальные профили значений HI, CO и пыли от центра аномалии до 2 радиусов (шаг — как для пыли).

    Посчитать: среднее, стандартное отклонение, минимум, максимум, 10%/90% перцентили по каждому компоненту (HI, CO, dust) внутри аномалии и в кольце border (например, R…1.2R).

2. Сравнить с контрольными зонами (фон):

    Повторить те же вычисления для случайных/контрольных точек (аналогично прошлому анализу по пыли).

3. Провести анализ по зонам:

    Отдельно для тонкого диска, толстого диска, гало.

4. Статистика морфологии:

    Сравнить радиальные профили и распределения по всем компонентам ISM:

        Совпадают ли пики, фронты, градиенты HI/CO/dust в аномалиях и фоновом фоне?

        Есть ли “переключение” фаз — например, где пыль спадает, газ нарастает (или наоборот)?

    Выделить аномалии с максимальным контрастом по каждому компоненту.

5. Итоговая проверка гипотезы:

    Если для всех компонент (HI, CO, dust) морфологические признаки (минимумы, градиенты, фронты) совпадают для аномалий, а не для контроля — это мощный аргумент в пользу фазовой самоорганизации среды.

    Если морфология разная, то среда устроена сложнее — требуется раздельный фазовый анализ по каждому компоненту.

Результаты:

    Радиальные профили HI, CO, пыли (по зонам, аномалиям и контролю).

    Гистограммы/boxplot контрастов по каждому компоненту.

    Корреляционные таблицы: совпадение фронтов и минимумов между компонентами ISM.

    Краткий отчёт с основными цифрами и визуализациями.
