Постановка задачи (с учётом всех имеющихся данных):

Исходные данные:
— Координаты (l, b), радиусы (r), S-оценки и зона принадлежности (диск/гало) для всех CMB-анoмалий (файлы уже загружены и обработаны).
— Классификация по масштабам (1°, 5°, 25°), а также статистика распределения аномалий по зонам.
— Теперь загружается (или доступна) карта пыли (Planck, EBV, SFD и др.).

Цель:
Проверить, проявляется ли фазовая самоорганизация среды в виде статистически значимой корреляции между CMB-анoмалиями и распределением пыли по всему небу, с учётом зон, масштабов и аномальности.

Задача:

    Для каждой CMB-анoмалии (с l, b, r):
    — вычислить среднее/медиану значения пыли внутри соответствующего радиуса r на карте пыли;
    — добавить результат в общую таблицу с уже имеющимися параметрами аномалии (S, зона, масштаб и т.д.).

    Для контроля:
    — аналогично рассчитать значения пыли для случайных/фоновых точек того же масштаба и зоны (по широте |b|).

    Анализ:
    — сравнить распределения “dust in anomalies” vs. “dust in control” для каждого масштаба и зоны;
    — посчитать медиану, стандартное отклонение, p-value (например, Mann–Whitney U-test).

    Дополнительно:
    — проверить, зависит ли сила аномалии (S-score) от уровня пыли;
    — визуализировать аномалии на фоне карты пыли.

Критерий:

    Если распределения отличаются статистически значимо (p < 0.05) — связь фазовых структур CMB и пыли подтверждается.

    Нет отличия — фазовая связь маловероятна для данного масштаба/зоны.

Ожидаемый результат:
— Итоговая таблица: для каждой аномалии — (l, b, r, S, зона, dust_value);
— Графики распределений, карта-overlay “dust + anomalies”, p-value статистики для всех масштабов и зон.
